# Способ через Podkop
Подразумевается что на текущем этапе у вас уже есть роутер с openWRT 24.10.5 - если нет, то вам [СЮДА](https://github.com/Ultraheal/great-win-of-the-horde/blob/main/openWRT-install.md)

Podkop несовместим со скриптом get-domains если он у вас установлен - [ЗДЕСЬ](https://podkop.net/docs/install/#nesovmestimost) вы можете прочитать, как его удалить

[Официальный сайт Podkop](https://podkop.net/)

Все инструкции есть на официальном сайте, ниже я постараюсь кратко описать базовый минимум, чтобы вы не копались по всем статьям там. Опишу подключение через amneziaWg, вариант с sing-box, который идет в инструкции по дефолту я не тестировал.

# Установка
Необходимо подключиться к роутеру по SSH и установить сам Podkop:

```sh <(wget -O - https://raw.githubusercontent.com/itdoginfo/podkop/refs/heads/main/install.sh)```

Далее устанавливаем инструменты для работы с AmneziaWG:

```sh <(wget -O - https://raw.githubusercontent.com/Slava-Shchipunov/awg-openwrt/refs/heads/master/amneziawg-install.sh)```

Нам нужны только сами пакеты для работы, на вопрос скрипта Do you want to configure the amneziawg interface? можете смело отвечать N

Далее в веб-интерфейсе роутера жмете System -> Reboot -> Perform reboot. Пока ждете перезагрузку переходите к этапу ниже.

# Развертывание Amnezia на собственном сервере
Подразумевается, что вы уже арендовали сервер на данном этапе

Поднятие AmneziaWG 2.0 достаточно простое, оно описано в официальном гайде на сайте амнезии https://docs.amnezia.org/ru/documentation/instructions/install-vpn-on-server админка будет встроена в их официальное приложение AmneziaVPN(это приложение доступно в Google Play вроде бы с любого региона, а вот в App Store оно недоступно в России, можно сменить регион например на Казахстан)

В приложении необходимо создать конфигурацию для вашего роутера. Заполняете поля, и в Connection format выбираете AmneziaWG native format. Далее жмете Share и передаете созданный файлик на свой компьютер.

Также можете расшарить доступ к своему VPN друзьям, семье и т.д. в этом же приложении. Просто создайте новую конфигурацию и в Connection format выберите For the AmneziaVPN app для использования через это же приложение, например на мобильных устройствах

# Настройка интерфейса Amnezia в openWRT
Переходите в веб-интерфейс роутера во вкладку Network -> Interfaces

Жмете Add new interface..., заполняете имя(например awg0) и выбираете протокол AmneziaWG VPN.

В открывшемся окне внизу есть кнопка Load configuration нажимаете и перетаскиваете файлик с конфигурацией, который создавали в пункте выше в это окно.

Переходим во вкладку Peers и нажимаем Edit. Находим опцию Persistent Keep Alive и вводим значeние 25. Без указания этого параметра туннель не соединяется с сервером.

Если ваша конфигурация содержала адреса DNS-серверов, то следует зайти во вкладку Advanced Settings и проверить поле Use custom DNS servers и удалить адреса серверов.

Нажимаем Save - Save - Save & Apply и после этого Restart рядом с только что созданным интерфейсом. Напротив строчек RX и TX должны появиться значения, отличные от 0.

# Настройка Podkop
После содеянного выше, возможно необходимо будет снова перезагрузить роутер, чтобы Podkop увидел новый интерфейс, но возможно это будет излишне - проверьте.

Переходим во вкладку Services -> Podkop

Во вкладке Sections выбираем Connection Type - VPN

Ниже в Network Interface только что созданный awg0

Ниже в Community Lists выбираете списки доменов которые будут ходить через туннель. У меня по дефолту стояло Russia Inside, к нему я добавил еще список Discord и Telegram

Можете выбрать User Domain List Type и User Subnet List Type если хотите к комьюнити спискам добавить свои кастомные домены и подсети.
Актуальный список доменов и подсетей можно подсмотреть [ЗДЕСЬ](https://github.com/itdoginfo/allow-domains) или [ЗДЕСЬ](https://iplist.opencck.org/). Там не всегда есть все для разблокировки, но почти всегда, список со временем обновляется. Либо можно уточнить по спискам у ребят [В ЧАТЕ itdog](https://t.me/itdoginfo)
Также у itdog есть великолепная [СТАТЬЯ](https://itdog.info/analiziruem-trafik-i-opredelyaem-domeny-kotorye-ispolzuyut-sajty-i-prilozheniya/) по тому как самостоятельно определять список доменов используемых сайтом или приложением

Для базовой настройки - это всë, жмете Save & Apply и у вас всë должно заработать

Если будут нужны какие-то более сложные и детальные настройки, на официальном сайте есть инструкции к каждой секции, для исключений доменов, перенаправления трафика устройства в туннель целиком и прочее
