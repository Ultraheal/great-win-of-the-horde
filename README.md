# Точечная маршрутизация по спискам доменов и подсетей
В данном гайде рассмотрим способы настройки точечной маршрутизации для роутеров на openWRT, в результате которой часть маршрутов мы сможем перенаправлять в VPN-туннель, а все остальное напрямую через провайдера

# Версии гайда
Первая версия расчитана на openWRT 23.05.5 и работу через скрипт get-domains, она более легковесная, но менее удобная в плане настройки(все нужно делать через терминал), 
также есть ряд технических ограничений с версией openWRT 23.05.5
(например нельзя простым способом установить amneziaWg с версией протокола 2.0 доступна только версия 1.5, технически установить возможно, 
но нужно собирать пакет из исходников самостоятельно - а кому оно надо), 
get-domains мной не тестировался на версии openWRT выше указанной, возможно оно будет работать, но утверждать не берусь, на 23.05.5 работает хорошо

Вторая версия расчитана на openWRT версии 24.10.5 и работу через пакет podkop, она занимает чуть больше места, но дает ui внутри интерфейса роутера, 
благодаря чему настройка становится более user friendly, а также с 24.10.5 мы получаем поддержку amneziaWg 2.0, где пакет просто ставится скриптом без танцев с бубном
